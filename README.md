# 🎨 GPTSkin — Customize ChatGPT Your Way
GPTSkin is a Chrome extension that lets you personalize your ChatGPT experience with custom colors, wallpapers, and themes.

## 🚀 Installation (Manual)

1. Clone or download this repository:
   
```bash
git clone https://github.com/YOUR_USERNAME/gptskin-extension.git
```

2. Open Chrome and go to:

```
chrome://extensions/
```

3. Enable **Developer Mode** (top right)

4. Click:
   👉 **Load unpacked**

5. Select the project folder

---

## 🎯 How to Use

1. Open ChatGPT
2. Click the **GPTSkin extension icon**
3. Choose one of the options:

### 🎨 Color Picker

* Select any color → applied instantly

### 🖼️ Upload Wallpaper

* Upload an image → becomes chat background

### 🌈 Quick Colors

* Choose from predefined aesthetic colors

### 💾 Save Theme

* Give your theme a name
* Save it to your collection

### 📂 My Collection

* Click saved themes to reapply
* Delete unwanted themes

### ❌ Reset

* Revert back to default ChatGPT theme

## ✨ Features

* 🎨 Apply custom background colors instantly
* 🖼️ Set your own wallpaper as chat background
* 💾 Save multiple themes (your personal collection)
* ⚡ Apply themes in real-time
* 🔁 Persistent themes (survive reloads)
* ❌ Reset to default anytime
* 🌈 Predefined aesthetic color options

---

## 🧠 How It Works

GPTSkin modifies the ChatGPT UI by:

* Applying background styles to the page
* Making internal chat containers transparent
* Persisting your theme using browser storage

## 🛠️ Tech Stack
